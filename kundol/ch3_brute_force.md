# 완전 탐색과 백트래킹

- 완전탐색, brute force: 완전히 탐색하는 것을 말한다
  - 앞에서부터 이미 해온 것이다

- 백트래킹: 완전 탐색을 하는 과정에서 불필요한 연산을 제거한 것
  - 상황 자체를 잘 이해해야 하는 것으로 보인다

- 원복: 상태값이 그 다음의 상태에 반영되지 않도록 하는 것
  - 문제에 따라서 굳이 이 행위를 할 필요가 없을 떄도 있다
