# 챕터 1

## 누적합(prefix sum), 구간 쿼리

prefix sum, suffix sum이 있는데, 기업형 코테에서는 prefix sum 정도면 충분하다   

아래 두 해 법이 생각나야 한다

- 부분합
- 펜윅트리

부분합: 정적 배열 (각 배열 요소가 변하지 않음)
펜윅트리: 동적 배열 (위와 달리 변함)
